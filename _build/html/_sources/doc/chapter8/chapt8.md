在使用 pandas 时，有时默认的配置方案并不能让我们舒服的进行数据分析。

幸运的是，pandas 也支持我们通过使用 `option` 和 `style` 相关功能进行 **自定义显示、样式**等个性化操作。

当然需要注意的是，这两个功能均未对原始数据产生影响，仅是在输出端进行相关配置！
